# CLAUDE.md — ととコーラ SEO 記事執筆ルール

このリポジトリで Claude Code が記事を生成するときに守るルール。
GitHub Actions で `prompts/article_generation.md` と一緒に読み込まれる。

## 1. ブランド

- 名称: ととコーラ
- サイト: https://totocola.com/
- 商品: クラフトコーラ シロップ 200ml
- 商品ページ: https://totocola.com/products/ととコーラ200ml
- コンセプト: **「夜にととのう、大人のクラフトコーラ」**
  - サウナの「ととのう」感覚を夜のリラックスタイムに重ねる
  - スパイス感・ハーバル感のある大人向けの味わい
  - カフェイン控えめで、就寝前でも楽しめる

## 2. ターゲット読者

- 30〜45 歳、男女
- 健康やウェルネスへの関心が高い
- サウナ・銭湯・ヨガ・キャンプ・ナチュラルワインなどのカルチャー圏
- 平日の夜、自宅で「ちょっといい時間」を作りたい人

## 3. トーン

- 落ち着いた、生活雑誌のような語り口（VERY/&Premium/dancyu のニュアンス）
- 押し売りせず、文化や背景を語ってから商品に触れる
- 一人称は使わない。客観 or やわらかな二人称（〜してみてください）
- 過度な感嘆符・絵文字は使わない（**絵文字禁止**）
- 文末は「です・ます調」で統一

### トーンサンプル（参考）

> 夜、台所の電気を半分だけ落として、氷の音を聞く。
> グラスに注いだクラフトコーラがゆっくりとほどけていく時間は、
> その日の緊張を少しずつ手放すための、小さな儀式のようなものです。

## 4. 記事構成のデフォルト

1. リード（150〜250 字）— 読者の状況を映す情景描写から入る
2. H2 セクション 4〜6 個
3. 各 H2 の下に H3 を 0〜2 個
4. 商品への自然な導線を本文中盤と末尾の 2 箇所に配置
5. まとめ（H2）
6. 文字数: 本文 2,500〜4,000 字目安

## 5. 必須・禁止事項

### 必須

- タイトルは 28〜34 字程度、検索キーワードを自然に含める
- meta description は 110〜140 字
- 主要キーワードを H1・最初の H2・本文冒頭に含める
- 記事末尾に商品ページ（https://totocola.com/products/ととコーラ200ml）への
  内部リンクを最低 1 本

### 禁止（薬機法・景表法）

以下は**断定表現で書かない**。`scripts/check_compliance.py` でも検出される。

- 「治る」「効く」「効果がある」「治療」「改善する」（病気・症状に対して）
- 「血圧が下がる」「血糖値が下がる」「免疫力が上がる」など医薬品的効能
- 「アンチエイジング」「若返る」「シミが消える」
- 「ダイエットできる」「痩せる」（特定保健用途食品でないため）
- 「No.1」「日本一」「最高」など根拠のない最上級表現
- 競合他社の名前を挙げての比較・批判

代わりに「〜と言われています」「〜と感じる人もいます」「研究が進んでいます」
など、出典・主観を明示する語り口に置き換える。

## 6. 出力フォーマット

`output/<slug>.md` と `output/<slug>.meta.json` の 2 ファイルを生成する。

### `<slug>.md`

```markdown
# 記事タイトル

リード文…

## H2 見出し1

本文…

…
```

### `<slug>.meta.json`

```json
{
  "title": "記事タイトル",
  "handle": "slug-in-kebab-case",
  "summary": "meta description (110〜140字)",
  "tags": ["タグ1", "タグ2", "タグ3"],
  "author": "ととコーラ編集部",
  "seo_title": "SEOタイトル (任意, 32字以内)",
  "seo_description": "SEO description (160字以内)"
}
```

`handle` は半角英数 + ハイフン。日本語スラッグは使わない。
重複しそうなら末尾に `-2` などを足す。

## 7. 競合調査と独自性

執筆前に WebSearch でキーワード上位 3〜5 件を確認し、

- 共通して扱われているサブトピックを把握
- それらをカバーした上で、ととコーラのブランド視点（夜・ととのう・大人）
  を加えて差別化する

検索意図が「商品比較」の場合は比較記事に寄せず、
「自宅でクラフトコーラを楽しむ夜の過ごし方」のような
ライフスタイル文脈で書き直す。
